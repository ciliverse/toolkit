# 文档

该目录包含项目的技术文档和说明文件。

## 文档类型

- **架构文档** - 系统架构设计和说明
- **API文档** - 接口文档和规范
- **部署文档** - 部署指南和配置说明
- **开发文档** - 开发环境搭建和编码规范
- **运维文档** - 运维手册和故障排除指南

## 文档格式

建议使用以下格式：
- Markdown (`.md`) - 主要文档格式
- OpenAPI/Swagger - API规范
- PlantUML - 架构图和流程图
- AsciiDoc - 复杂文档

## 文档维护

- 保持文档与代码同步更新
- 使用版本控制管理文档变更
- 定期审查和更新过时内容
